一些自己记笔记用到的工具

  • Windows/Linux + Vim + Firefox，以及 @

    一些小工具：

    • district10/pin: 保持窗口在上/总在最前（make window always on top）
    • district10/AltWindowDrag: Drag window by holding Alt key and dragging it, works on Windows just like on Linux (A personal backup), Windows 上实现 Linux 的【按住 Alt 拖动窗口】的实用功能
  • district10/jwatch: watch directory for changes, written in Java @

    写博客/笔记的时候，不用再手动 make。

    不再需要 Markdown 编辑器，用 Vim 写就好，然后把浏览器放在对边。推荐安装浏览器插件来 auto-reload，比如 Firefox 的 Auto Reload :: Add-ons for Firefox。（Chrome 上应该也有类似的工具。）
